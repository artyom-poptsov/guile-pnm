* Guile-PNM
A GNU Guile library for working with [[https://en.wikipedia.org/wiki/Netpbm][PNM]] format family (PBM, PGM, PPM.)

** Installation
*** GNU Guix
 #+BEGIN_EXAMPLE shell
$ guix install guile-pnm
 #+END_EXAMPLE

Development version from =guix.scm=:
#+BEGIN_EXAMPLE shell
$ guix build -f guix.scm
$ guix package -f guix.scm
#+END_EXAMPLE

*** Manual
 #+BEGIN_EXAMPLE shell
$ git clone https://github.com/artyom-poptsov/guile-pnm.git
$ cd guile-pnm
$ autoreconf -vif
$ ./configure
$ make -j$(nproc)
$ sudo make install
#+END_EXAMPLE
